# Chatbot para tutoría Académica Personalizada
AI-powered tutoring system for secondary education using RAG architecture (FastAPI &amp; React).

## Project Overview
This research project explores the implementation of a **Retrieval-Augmented Generation (RAG)** architecture to provide personalized academic tutoring for 5th-grade secondary students in Peru. The goal is to reduce AI hallucinations and align responses with the National Curriculum.

## Technical Stack
- **Backend:** FastAPI (Python)
- **Frontend:** React.js
- **AI Models:** OpenAI GPT-4o / o1-preview
- **Vector Database:** (e.g., Pinecone / ChromaDB)
- **Orchestration:** LangChain / LlamaIndex

## Research Objectives
- Evaluate the influence of RAG on student cognitive performance.
- Measure the accuracy of pedagogical feedback in a public school setting.
- Analyze the socio-emotional impact of AI-driven tutoring.
